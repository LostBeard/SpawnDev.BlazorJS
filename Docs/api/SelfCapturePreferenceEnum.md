# SelfCapturePreferenceEnum

**Namespace:** `SpawnDev.BlazorJS.JSObjects`  
**Source:** `JSObjects/SelfCapturePreferenceEnum.cs`

## Values

|---|---|---|
| `Include` | `"include"` | The application prefers the surface be included among the choices offered. |
| `Exclude` | `"exclude"` | The application prefers the surface be excluded from the choices offered. |
