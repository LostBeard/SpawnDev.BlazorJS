# DataView

**Namespace:** `SpawnDev.BlazorJS.JSObjects`  
**Inherits:** `JSObject`  
**Source:** `JSObjects/DataView.cs`  
**MDN Reference:** [DataView on MDN](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/DataView)

> The DataView view provides a low-level interface for reading and writing multiple number types in a binary ArrayBuffer, without having to care about the platform's endianness. https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/DataView
