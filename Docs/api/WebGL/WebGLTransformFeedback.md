# WebGLTransformFeedback

**Namespace:** `SpawnDev.BlazorJS.JSObjects`  
**Inherits:** `WebGLObject`  
**Source:** `JSObjects/WebGL/WebGLTransformFeedback.cs`  
**MDN Reference:** [WebGLTransformFeedback on MDN](https://developer.mozilla.org/en-US/docs/Web/API/WebGLTransformFeedback)

> The WebGLTransformFeedback interface is part of the WebGL 2 API and enables transform feedback, which is the process of capturing primitives generated by vertex processing. It allows to preserve the post-transform rendering state of an object and resubmit this data multiple times. https://developer.mozilla.org/en-US/docs/Web/API/WebGLTransformFeedback
