## Example

```csharp
// Get SubtleCrypto from the Crypto global
using var crypto = new Crypto();
using var subtle = crypto.Subtle;

// Generate an ECDSA key pair for signing/verification
using var keyPair = await subtle.GenerateKey<CryptoKeyPair>(
    new EcKeyGenParams { Name = "ECDSA", NamedCurve = "P-256" },
    extractable: true,
    keyUsages: new[] { "sign", "verify" }
);

// Sign some data with the private key
byte[] message = System.Text.Encoding.UTF8.GetBytes("Hello, WebCrypto!");
using var signature = await subtle.Sign(
    new EcdsaParams { Hash = "SHA-256" },
    keyPair.PrivateKey!,
    message
);
Console.WriteLine($"Signature size: {signature.ByteLength} bytes");

// Verify the signature with the public key
bool isValid = await subtle.Verify(
    new EcdsaParams { Hash = "SHA-256" },
    keyPair.PublicKey!,
    signature,
    message
);
Console.WriteLine($"Signature valid: {isValid}");

// Export the public key in SPKI format
using var exportedKey = await subtle.ExportKeySpki(keyPair.PublicKey!);
byte[] publicKeyBytes = exportedKey.ReadBytes();

// Compute a SHA-256 digest
using var digest = await subtle.Digest("SHA-256", message);
byte[] hash = digest.ReadBytes();
Console.WriteLine($"SHA-256 hash length: {hash.Length} bytes");
```
