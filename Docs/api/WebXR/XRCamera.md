# XRCamera

**Namespace:** `SpawnDev.BlazorJS.JSObjects`  
**Inherits:** `JSObject`  
**Source:** `JSObjects/WebXR/XRCamera.cs`  

> The XRCamera interface is introduced as a way to expose information about the camera texture that can be obtained from XRWebGLBinding. https://immersive-web.github.io/raw-camera-access/#xr-camera-section

## Properties

| Property | Type | Access | Description |
|---|---|---|---|
| `Width` | `long` | get | The XRCamera contains width attribute that contains the width (in texels) of the camera image. |
| `Height` | `long` | get | The XRCamera contains height attribute that contains the height (in texels) of the camera image. |
