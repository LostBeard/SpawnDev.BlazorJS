# MediaStreamAudioDestinationNode

**Namespace:** `SpawnDev.BlazorJS.JSObjects`  
**Inherits:** `AudioNode`  
**Source:** `JSObjects/WebAudio/MediaStreamAudioDestinationNode.cs`  
**MDN Reference:** [MediaStreamAudioDestinationNode on MDN](https://developer.mozilla.org/en-US/docs/Web/API/MediaStreamAudioDestinationNode)

> The MediaStreamAudioDestinationNode interface represents an audio destination consisting of a WebRTC MediaStream with a single AudioMediaStreamTrack, which can be used in a similar way to a MediaStream obtained from getUserMedia(). https://developer.mozilla.org/en-US/docs/Web/API/MediaStreamAudioDestinationNode

## Properties

| Property | Type | Access | Description |
|---|---|---|---|
| `Stream` | `MediaStream` | get | A MediaStream containing a single AudioMediaStreamTrack with the same number of channels as the node itself. |
