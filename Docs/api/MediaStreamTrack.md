# MediaStreamTrack

**Namespace:** `SpawnDev.BlazorJS.JSObjects`  
**Inheritance:** `JSObject` -> `EventTarget` -> `MediaStreamTrack`  
**MDN Reference:** [MediaStreamTrack](https://developer.mozilla.org/en-US/docs/Web/API/MediaStreamTrack)  
**Transferable:** Yes (`[Transferable(TransferRequired = true)]`)

> The `MediaStreamTrack` class represents a single media track within a stream - typically an audio or video track, but other track types may exist. Tracks are obtained from `MediaStream.GetTracks()`, `MediaStream.GetVideoTracks()`, or `MediaStream.GetAudioTracks()`, and originate from `getUserMedia()`, `getDisplayMedia()`, or `RTCPeerConnection` incoming tracks.

## Constructors

| Signature | Description |
|-----------|-------------|
| `MediaStreamTrack(IJSInProcessObjectReference _ref)` | Deserialization constructor. |

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `Id` | `string` | Read-only. A unique identifier (GUID) for the track, generated by the browser. |
| `Kind` | `string` | Read-only. `"audio"` if the track is an audio track, `"video"` if it is a video track. Does not change if the track is disassociated from its source. |
| `Label` | `string` | Read-only. A user agent-assigned label identifying the track source (e.g., `"internal microphone"`, `"FaceTime HD Camera"`). Empty until a source is connected. |
| `Enabled` | `bool` | `true` if the track is enabled and rendering media; `false` if disabled (silence/black). Use this for mute/unmute functionality. |
| `Muted` | `bool` | Read-only. `true` if the track is unable to provide media data due to a technical issue (not user-initiated). |
| `ReadyState` | `string` | Read-only. `"live"` - the input is connected and providing real-time data. `"ended"` - the input is no longer providing data. |
| `ContentHint` | `string` | A hint for how the track content should be treated by API consumers. |

## Methods

| Method | Return Type | Description |
|--------|-------------|-------------|
| `Stop()` | `void` | Stops the track and disassociates it from its source. The track state becomes `"ended"`. |
| `Clone()` | `MediaStreamTrack` | Creates a duplicate of the track with a new unique `Id`. |
| `GetSettings()` | `MediaTrackSettings` | Returns the current values of all constrainable properties (width, height, frameRate, sampleRate, etc.). |
| `GetSettings<T>()` | `T` | Returns settings as a custom type that extends `MediaTrackSettings`. |
| `GetConstraints()` | `MediaTrackConstraints` | Returns the constraints most recently applied via `ApplyConstraints()`. |
| `GetCapabilities()` | `MediaTrackCapabilities` | Returns the range of constrainable property values the track source supports. |
| `ApplyConstraints(MediaTrackConstraints constraints)` | `Task` | Applies constraints to the track (e.g., resolution, frame rate). Rejects with `OverconstrainedError` if unsatisfiable. |
| `ApplyConstraints()` | `Task` | Clears all currently set custom constraints. |

## Events

| Event | Type | Description |
|-------|------|-------------|
| `OnEnded` | `ActionEvent<Event>` | Fired when playback or streaming has stopped because the end of media was reached or no further data is available. |
| `OnMute` | `ActionEvent<Event>` | Fired when the track's source is temporarily unable to provide media data. |
| `OnUnMute` | `ActionEvent<Event>` | Fired when the track's source is once again able to provide media data after a period of being muted. |

## Example

```csharp
// Get a video track from user media
using var navigator = JS.Get<Navigator>("navigator");
using var mediaDevices = navigator.MediaDevices;
using var stream = await mediaDevices.GetUserMedia(new MediaStreamConstraints
{
    Video = new MediaTrackConstraints
    {
        Width = new ConstrainULong { Ideal = 1920 },
        Height = new ConstrainULong { Ideal = 1080 },
        FrameRate = new ConstrainDouble { Ideal = 30 }
    },
    Audio = true
});

// Inspect video track
using var videoTrack = stream.GetFirstVideoTrack();
if (videoTrack != null)
{
    Console.WriteLine($"Track ID: {videoTrack.Id}");
    Console.WriteLine($"Kind: {videoTrack.Kind}");        // "video"
    Console.WriteLine($"Label: {videoTrack.Label}");       // e.g., "FaceTime HD Camera"
    Console.WriteLine($"ReadyState: {videoTrack.ReadyState}"); // "live"
    Console.WriteLine($"Enabled: {videoTrack.Enabled}");    // true
    Console.WriteLine($"Muted: {videoTrack.Muted}");        // false

    // Read current settings
    var settings = videoTrack.GetSettings();
    Console.WriteLine($"Resolution: {settings.Width}x{settings.Height}");
    Console.WriteLine($"Frame rate: {settings.FrameRate}");

    // Check capabilities
    var capabilities = videoTrack.GetCapabilities();
    Console.WriteLine($"Max resolution: {capabilities.Width?.Max}x{capabilities.Height?.Max}");

    // Apply new constraints at runtime
    await videoTrack.ApplyConstraints(new MediaTrackConstraints
    {
        Width = new ConstrainULong { Ideal = 1280 },
        Height = new ConstrainULong { Ideal = 720 }
    });

    // Mute/unmute via Enabled property
    videoTrack.Enabled = false;  // Video goes black (muted)
    videoTrack.Enabled = true;   // Video resumes
}

// Handle track lifecycle events
using var audioTrack = stream.GetFirstAudioTrack();
if (audioTrack != null)
{
    audioTrack.OnEnded += (evt) =>
    {
        Console.WriteLine("Audio track ended");
        evt.Dispose();
    };

    audioTrack.OnMute += (evt) =>
    {
        Console.WriteLine("Audio track muted (technical issue)");
        evt.Dispose();
    };

    audioTrack.OnUnMute += (evt) =>
    {
        Console.WriteLine("Audio track unmuted");
        evt.Dispose();
    };

    // Clone the track for use in a second context
    using var audioClone = audioTrack.Clone();
    Console.WriteLine($"Clone ID: {audioClone.Id}"); // Different from original

    // Stop the track when done
    audioTrack.Stop();
    Console.WriteLine($"ReadyState: {audioTrack.ReadyState}"); // "ended"
}
```
