# VideoFacingModeEnum

**Namespace:** `SpawnDev.BlazorJS.JSObjects`  
**Source:** `JSObjects/VideoFacingModeEnum.cs`  

> The VideoFacingModeEnum enum of the Media Statistics API is used to describe the facing mode of a video source.

## Values

| Name | JSON Value | Description |
|---|---|---|
| `User` | `"user"` | The source is facing toward the user (a self-view camera). |
| `Environment` | `"environment"` | The source is facing away from the user (viewing the environment). |
| `Left` | `"left"` | The source is facing to the left of the user. |
| `Right` | `"right"` | The source is facing to the right of the user. |
