# GPUCompilationMessage

**Namespace:** `SpawnDev.BlazorJS.JSObjects`  
**Inherits:** `JSObject`  
**Source:** `JSObjects/WebGPU/GPUCompilationMessage.cs`  
**MDN Reference:** [GPUCompilationMessage on MDN](https://developer.mozilla.org/en-US/docs/Web/API/GPUCompilationMessage)

> The GPUCompilationMessage interface of the WebGPU API represents a single informational, warning, or error message generated by the GPU shader module compiler. An array of GPUCompilationMessage objects is available in the messages property of the GPUCompilationInfo object accessed via GPUShaderModule.getCompilationInfo(). https://developer.mozilla.org/en-US/docs/Web/API/GPUCompilationMessage

## Properties

| Property | Type | Access | Description |
|---|---|---|---|
| `Length` | `int` | get | A number representing the length of the substring that the message corresponds to. |
| `LineNum` | `int` | get | A number representing the line number in the shader code that the message corresponds to. |
| `LinePos` | `int` | get | A number representing the position in the code line that the message corresponds to. This could be an exact point, or the start of the relevant substring. |
| `Message` | `string` | get | A string representing human-readable message text. |
| `Offset` | `int` | get | A number representing the offset from the start of the shader code to the exact point, or the start of the relevant substring, that the message corresponds to. |
| `Type` | `string` | get | An enumerated value representing the type of the message - "error", "info", or "warning". |
